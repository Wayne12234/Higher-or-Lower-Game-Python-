# Higher-or-Lower-Game-Python-
A simple command-line Higher or Lower game built in Python. The player compares two randomly selected people and guesses who has more followers. The score increases for each correct guess and the game ends when a wrong answer is given. A beginner-friendly project using functions, loops, and dictionaries.

## How the Game Works

1. Two random people are selected from a dataset.
2. The player is asked to guess who has more followers by typing **A** or **B**.
3. If the guess is correct, the score increases.
4. If the guess is wrong, the game ends and the final score is displayed.

## Features

* Random selection using Python's `random` module
* Score tracking
* Input validation
* Simple and beginner-friendly Python logic

## Requirements

* Python 3.x

## How to Run

```bash
python higher_lower_game.py
```
